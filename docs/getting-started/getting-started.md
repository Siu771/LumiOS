# Getting Started

{% stepper %}
{% step %}
### Clone the Repository

```shellscript
git clone https://github.com/LuminesenceProject/LumiOS.git
cd LumiOS
```
{% endstep %}

{% step %}
### Install Dependencies

```shellscript
npm install
```
{% endstep %}

{% step %}
### Run the Development Server

```shellscript
npm run dev
```
{% endstep %}

{% step %}
### Build for Production

```shellscript
npm run build
```
{% endstep %}
{% endstepper %}

Alternatively, you can click the branch dropdown and download a specified version there.
